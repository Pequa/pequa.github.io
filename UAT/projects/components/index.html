Price Box

Project Role: Director & Designer

Since SimplePart's platform contains millions of products across thousands of sites, with billions of page views each year, each component that we design needs to be fully modular, responsive, and inherit styling from whatever site it lives on. The price box on the product page is one of the first components I redesigned as a part of an effort to componentize each element on our platform, containing flexible elements like fulfillment option slots, stock and warranty information, and reviews, all of which can populate in any combination depending on what data needs to return.

Product Information Tabs

Project Role: Director & Designer

As we added more and more functionality to the product page over the years, what started out as a small set of horizontal tabs to display supplemental information quickly became more and more crowded. As such, I redesgined the container to be able to support this information in a more modular, responsive layout. Each tab was redesigned in a more consistent manner based on the data that would populate within, so that tabs that functioned similarly had a similar look and feel, all allowing any supplemental information to be displayed in a clean, readable manner.

Dealer Finder

Project Role: Director

This component initially arose as part of implementing the ability to check out on our parent-level sites, as traditionally dealer sites were the only sites on our platform that had e-commerce enabled. The widget was designed to appear on any page, and present the nearest dealers that can fulfill the order to the user based on geolocation. To help users find alternate dealers that may have different business offerings to fit their needs, there are multiple slots for payment and fulfillment options, ratings, and more, in case the user wishes to override the local dealer.

Guided Navigation

Project Role: Director & Designer

Guided Navigation, as it became known, was essentially proposed as an effort to reduce bounce rate for users that landed on our product pages with incomplete fitment, and it succeeded in doing so. It provides the ability to confirm or deny fitment of the part they're looking at based on their vehicle, as well as recommend alternate products if the part is not what they are looking for. It needed to be eye catching, and help "guide" the user to the parts they were initially searching for, hence the name.
